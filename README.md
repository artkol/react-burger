# React Burger App

Learning react/redux project

## Built with

* create-react-app
* react-16
* react-router
* redux
* firebase api
* jest/enzyme
* localStorage
* css modules

## Installation and Running

```
git clone https://github.com/artkol/react-burger
npm install
npm start

http://localhost:3000/
```
